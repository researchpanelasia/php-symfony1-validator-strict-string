# NAME

sfValidatorStrictString - More strict string validation for Symfony 1.x

# SYNOPSIS

Use this just like `sfValidatorString`.

# DESCRIPTION

This validator detects characters `[\r\n\s\t]` in prefix and suffix of a string.
